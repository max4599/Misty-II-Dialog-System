# Misty-II-Dialog-System
Interactive dialog and face recognition system for the Misty II social robot.

Features
	•	Face recognition
	•	Voice interaction
	•	Event-driven programming
	•	Interactive conversations
	•	Person-specific reactions

Technologies
	•	Python
	•	Misty II API

Description

This project implements an interactive dialog system for the Misty II robot. The robot can recognize people, react differently to specific users, start conversations through voice commands and follow predefined dialogue flows based on user responses.
